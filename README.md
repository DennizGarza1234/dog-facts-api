# Dog Facts API

A simple REST API built with Node.js and Express.js that serves fun dog facts in JSON format.

---

## Features

- GET `/facts` - Returns all dog facts or a specified number.
- GET `/facts/random` - Returns a single random dog fact.
- Handles invalid requests with clear error messages.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/dog-facts-api.git
cd dog-facts-api
```
2. Install the dependencies:
```bash
npm install
```
3. Start the server:
```bash
node server.js
```
